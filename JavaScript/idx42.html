JavaScript: Loops and Strings

1. For Loop

The for loop is used when you know how many times to repeat an action.

for (let i = 1; i <= 5; i++) {
    console.log("Number:", i);
}

2. While Loop

The while loop repeats as long as a condition is true.

let i = 1;
while (i <= 5) {
    console.log("Count:", i);
    i++;
}

3. Do-While Loop

The do...while loop executes the block at least once before checking the condition.

let j = 1;
do {
    console.log("Value:", j);
    j++;
} while (j <= 5);

4. String Operations

Common operations you can perform on strings.

let text = "JavaScript";
console.log("Length:", text.length);
console.log("Uppercase:", text.toUpperCase());
console.log("Lowercase:", text.toLowerCase());
console.log("Substring (0-4):", text.substring(0, 4));
console.log("Replace:", text.replace("Java", "Type"));
